# Animal Face Test (AI)

This project is a simple, AI-powered "Animal Face Test" developed in Firebase Studio. It uses Teachable Machine models to identify whether you look more like a dog or a cat.

## 🚀 Deployment

The site is deployed via **GitHub Pages**.

- **URL:** [https://chris850123.github.io/test/](https://chris850123.github.io/test/)

### How to access:
1. Ensure the **GitHub Pages** settings on the repository are set to the `main` branch.
2. Visit the URL above.

## 🛠️ Features
- AI image recognition (Dog vs. Cat)
- Responsive UI
- Partnership form
- Comment section (Disqus)
